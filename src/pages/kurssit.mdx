# Kurssit

Yleiskatsaus tarjolla oleviin ohjelmointiväyliin ja -kursseihin.   
Kurssit painottuvat ohjelmoinnissa Javascript tai Python kieliin. Kurssi moduuleilla on tarkoitus tukea valitsemaa väylää ja tarjota projektin toteutus mahdollisuus.  
Ohjelmointiväylien nimen perään on merkitty niiden vaikeusaste. Jokaisen polun sisältä löytyy myös omat vaikeusasteensa, näin jokainen väylä pysyy edistyvästi myös haasteellisena.

### Muistutus tehtäväosuuksien suorittamisesta
Kun teet tehtävä osuuksia niin halutessasi voit jättää kyseisen tehtäväosuuden alaosaan reaktion tai sitten kommentin aiheesta. Näin autat sivuston kehittämisessä ja palautteen saamisessa. Sivusto kehittyy saadun palautteen myötä. 

### Parannus- ja korjausehdotukset
Jos löydät sivustolta virheitä, haluat antaa palautetta tai muuten vaan kommentoida sivun sisältöä, niin se onnistuu jokaisen sivun alta löytyvällä reaktio nappulalla tai sitten kirjoittamalla suoraan kommentin kommenttikenttään.
Tarkempaa palautetta voi jättää GitHub-repoon osoitteessa: https://github.com/eync/ohjelmointikurssit/issues
