Javascript

Javascript on web-ympäristössä pyöritettävä skriptikieli. Alkuperäinen tarkoitus kielelle oli tuoda lisätoiminallisuutta verkkosivuille. Sitä voidaan käyttää esimerkiksi: HTML-pohjaisten verkkosivujen dokumenttipuun manipulointiin, pyörittää palvelinpuolen verkko-ohjelmoinnissa Node.js avustuksella ilman tarvetta erilliselle verkkoselaimelle, mobiilikehityksessä ja pelikehityksessä.

Kirjastot​

Vaikka javascriptillä itsessään voi saada paljon aikaiseksi, niin samaan aikaan javascript kirjastojen ja kehysohjelmistojen suosio on noussut ylöspäin.
Tässä muutama tärkeä Javascript kirjasto:

  1. React.js
  2. Node.js
  3. Vue.js
  4. jQuery
Huomioitavaa

jQuery on otettu listalle koska se on mainitsemisen arvoinen ja yhä suuri jättiläinen Javascript markkinoilla. Vaikka sen onkin jo moni yrittänyt korvata muilla uudemmilla ja paremmilla kirjastoilla tai kielillä, niin on se onnistunut pitämään pintansa. Sillä onkin yhä viikottainen latausmäärä n. 4+ miljoonaa kertaa (Lähde).
Toivottavaa onkin että kaikki siirtyisivät käyttämään parempia vaihtoehtoja kuten esimerkiksi natiivia javascriptiä, Vue.js- tai React.js kirjastoa.
2021 oli selkeästi muutoksen vuosi ja tällöin jQuery tipahti toiselle sijalle web-kehittäjille tehdyssä kyselyssä, React.js ottaessa ensimmäisen paikan (Lähde).
Kurssilla ei tulla panostautumaan kyseiseen Javascript-kirjastoon lainkaan.

Document Object Model (DOM) standardi​

Luodessasi HTML-dokumenttiä sisällytät siihen valitsemasi HTML-merkinnät ja -elementit. Näin tehdessäsi luot tietynlaisen hierarkian jota kutsutaan myös puuksi (engl. tree) (vaihtoehtoisesti: dokumentti puu [engl. document tree]).

  • HTML esimerkki
  • DOM puu
  • Selaimen Javascript Objekti
index.html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body>

</body>

</html>
Kuinka verkkoselain näkee HTML-dokumentin
    ├─DOCTYPE: html
    └─HTML
        ├─HEAD
        │    └─TITLE
        └─BODY
Kuvassa näet DOM puu:n ominaisuudet.
