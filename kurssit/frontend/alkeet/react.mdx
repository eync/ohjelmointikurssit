React on avoimeen koodilähteeseen perustuva front-end Javascript kirjasto. Sillä voidaan rakentaa käyttöliittymiä, komponentteja ja erilaisia modulaarisia osia front-end järjestelmään. Koodia voidaan pilkkoa pienempiin osiin, jolloin koodia on helpompi hallita. Kielen kehittäjinä toimivat Facebook ja React-yhteisö.  

## Virtual DOM
React hyödyntää Virtual DOM tekniikkaa. Ideana on että React rakentaa virtuaalisen DOM:n joka seuraa koodissa tapahtuvia muutoksia, ja tekee ainoastaan tarpeelliset muutokset sitten oikeaan DOM:iin.

## Käyttöliittymän rakentaminen
React on juurikin tehty käyttöliittymän rakentamista varten, siinä on todella monia mahdollisuuksia toteuttaa eri alustoille olevia käyttöliittymä projekteja. 

:::info 

Valmiita käyttöliittymä- ja komponentti kirjastoja, mitä voit käyttää omissa projekteissa.
- Material UI
- Semantic UI
- Chakra UI 
- Ant Design
:::
## Koodin lajittelu
Kun projektin koodi määrä kasvaa, niin on normaalia että koodi myös hidastuu. 



## Create-React-App
Create React App (lyh. CRA) on Facebookin luoma työkalu joka nopeuttaa applikaation konstruktoimista ja säästää aikaa react projektin konfiguroimiselta. Pelkällä yhdellä komennolla saat nopean työkalun käyttöösi projektia varten.
